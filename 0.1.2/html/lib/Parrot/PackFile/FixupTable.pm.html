NAME

Parrot::PackFile::FixupTable - A Parrot Fix-Up Table

SYNOPSIS 

  use Parrot::PackFile::FixupTable;

DESCRIPTION

Instances of Parrot::PackFile::FixupTable are used to represent fixup tables in Parrot packfiles.

Returns a new instance.

Instance Methods

clear()

Does nothing.
