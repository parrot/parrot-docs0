=head1 NAME

README - Readme file for Parrot's 'parrot-docs0' documentation repository.

=head1 DESCRIPTION

The purpose of this README is to record the fact that this repository contains 
the documentation for version 0.0.6 (both of them) through version 0.9.1 of 
Parrot.  In short, this repository is, in-the-main, about maintaining a set of 
all of Parrot's documentation on github.
